# Budget Planner

A simple weekly budget planner web app.

## Folder Structure

The workspace is organized by project type:

- `web`: the Budget Planner static website
- `java/src`: Java source files
- `.vscode`: VS Code workspace settings and launch configurations

Java compiled output is generated in the `bin` folder by default.

## Run the Website

Open `web/index.html` in a browser, or use the VS Code launch configuration for Opera or Chrome.

## Java Notes

The Java placeholder app is in `java/src/App.java`. If Java dependencies are needed later, place `.jar` files under `java/lib`.
